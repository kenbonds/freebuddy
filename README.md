# FreeBuddy AI Tool

### 你的本地 AI 全能工坊 · 桌面办公 · 软件开发 · 网页部署

> 一款真正属于你自己的 AI 工作站：写文档、做软件、搭网站、部署上线，全部在你的电脑上完成。无需联网、无需付费、无需上传隐私数据。

## 📖 项目简介

FreeBuddy AI Tool 不是又一个在线 AI 聊天窗口。它是一个**完全运行在你本地的 AI 多智能体工程平台**，能像一支完整的技术团队一样替你干活：

| 你能用它做什么 | 具体场景 |
|---|---|
| 📄 **桌面办公** | AI 写报告、润色公文、整理会议纪要、翻译跨国邮件、生成 PPT 大纲 |
| 💻 **软件开发** | AI 生成代码、Debug 调试、性能优化、代码审查、架构设计建议 |
| 🌐 **网页搭建与部署** | AI 写前端页面、搭后端接口、生成 Nginx 配置、自动化部署脚本 |
| 🔁 **全流程自动化** | 从需求→架构→编码→测试→部署→归档，AI 多智能体流水线自动流转 |

### 和 ChatGPT / Claude 网页版有什么不同？

| | 在线 AI 网页 | FreeBuddy |
|---|---|---|
| 🔒 **隐私** | 你的代码/文档要上传到别人服务器 | **全部本地运行，数据不出电脑** |
| 💰 **费用** | 按月订阅或用额度 | **完全免费，不限次数** |
| 🌐 **联网** | 必须有网络 | **Ollama 离线可用，无网照干** |
| 📦 **工程化** | 聊完就没了，没法追踪 | **工单流转、版本归档、审计日志全都有** |

- **前端**：React + TypeScript + Ant Design

- **工程规范**：TypeScript 严格模式、零 any、标准化目录结构

## 🚀 快速开始

### 环境要求
- Node.js 16+
- npm / yarn
- 可选：Ollama（离线 AI 能力）

### 安装运行

```bash
git clone https://github.com/kenbonds/freebuddy.git
cd freebuddy
npm install
npm run start
```

浏览器打开 `http://localhost:3000` 即可开始使用。

### 配置 AI 模型

**本地 Ollama（推荐，免费离线）：**
```bash
# 安装 Ollama https://ollama.com/
ollama pull llama3
# 项目默认适配，无需任何配置
```

**远程 OpenAI：**
进入模型配置页面，填入你的 OpenAI Key，一键切换即可。

### 打包部署
```bash
npm run build
```
